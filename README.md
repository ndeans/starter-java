# java-starter

A minimal Java 21 Maven starter project. Used as the scaffold for Raven-Jobs.

## Stack

- Java 21
- SLF4J + Logback (logging)
- JUnit Jupiter 5 (testing)

## Building

```
mvn compile
```

## Testing

```
mvn test
```

## Running

```
mvn exec:java -Dexec.mainClass="com.example.App"
```
